# Simulador de Orçamento - Freelancer TI

Projeto desenvolvido para a disciplina de Desenvolvimento Web.

A ideia é simular a contratação de serviços de TI. O cliente acessa a página, seleciona os serviços que quer contratar e vê o valor total sendo atualizado na hora.

## Como usar

Só abrir o arquivo `index.html` no navegador. Não precisa instalar nada.
